# Data Analytics Project

## Overview

This project demonstrates a complete Data Analytics workflow, starting from data collection to business insights. The process includes data cleaning, exploratory data analysis (EDA), SQL analysis, dashboard creation in Power BI, report generation, and a final presentation. The goal is to transform raw data into meaningful insights that support data-driven decision-making.

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Loaded data using the Pandas library.
* Performed an initial inspection of rows, columns, and data types.

### 2. Exploratory Data Analysis (EDA)

* Examined dataset structure and summary statistics.
* Identified missing values and duplicate records.
* Explored relationships between variables.
* Created visualizations to identify trends and patterns.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data formats.
* Prepared the dataset for analysis.

### 4. SQL Analysis (MySQL)

* Imported the cleaned dataset into MySQL.
* Executed SQL queries to:

  * Filter records
  * Aggregate data
  * Perform grouping and sorting
  * Generate business insights
  * Analyze key performance metrics

### 5. Power BI Dashboard

Created an interactive dashboard including:

* KPI Cards
* Sales/Performance Trends
* Category Analysis
* Regional Analysis
* Interactive Filters (Slicers)
* Charts and Visualizations

### 6. Report Creation

Prepared a detailed analytical report summarizing:

* Data cleaning process
* Key findings
* Business insights
* Recommendations

### 7. Presentation (Gamma)

Created a professional presentation highlighting:

* Project overview
* Methodology
* Analysis process
* Dashboard screenshots
* Key insights
* Final conclusions

## How to Run

### Prerequisites

* Python 3.x
* Jupyter Notebook or VS Code
* MySQL Server
* Power BI Desktop

### Steps

1. Clone the repository.
2. Install the required Python libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Open the Jupyter Notebook and run all cells.
4. Import the cleaned dataset into MySQL.
5. Execute the SQL queries in the `queries.sql` file.
6. Open the Power BI (`.pbix`) file to explore the interactive dashboard.
7. Review the project report and presentation for a complete summary.
